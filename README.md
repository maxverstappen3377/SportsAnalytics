# Badminton Performance Analytics System

A comprehensive descriptive, predictive, and prescriptive analytics platform for badminton, mapping raw video to post-match tactical coaching recommendations.

## Tech Stack
- **Backend**: Python 3.14, FastAPI, Postgres (pgvector), Redis + Celery
- **Machine Learning**: Scikit-Learn (XGBoost/Random Forest), PyTorch (LSTM sequence forecasting)
- **Frontend**: Next.js 14, Recharts, Tailwind CSS
